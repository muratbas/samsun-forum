# Project Brief: Samsun Forum

## Project Overview
A hyper-local, Turkish-language forum website exclusively for Samsun, Turkey residents. This is a school project designed to create the central digital "meydan" (public square) for Samsun, replacing chaotic social media groups with an organized community platform.

## Core Objectives
1. Create a Reddit-like forum focused exclusively on Samsun city
2. Enable residents to share posts (text + images) about local topics
3. Foster community engagement through voting and discussions
4. Provide event discovery for Samsun residents
5. Maintain a safe, moderated environment for local conversations

## Primary Features (MVP)
- **User Authentication**: Google login → nickname selection → nickname/password login
- **Posts**: Text and image posts in admin-defined categories
- **Voting System**: Reddit-style upvote/downvote
- **Topics/Categories**: Admin-managed categories (e.g., Yemek, Etkinlik, Samsunspor, Trafik)
- **Feed Views**: Popular, New, Most Voted sorting
- **Public Reading**: Anyone can view, login required to post/vote

## Secondary Features (Future)
- **Events Section**: Upcoming events in Samsun (API/scraping integration)
- **Automated Moderation**: Temp ban for severe profanity
- **Admin/Moderator System**: Content management and user moderation

## Target Audience
- Samsun residents (all ages)
- Initial scale: ~500 users/day
- Turkish-speaking community members

## Success Criteria
- Functional forum with posts, voting, and categories
- Clean, modern UI matching provided design
- Fast performance and mobile responsiveness
- Safe, moderated community space
- Successfully deployed and accessible

## Constraints
- School project timeline
- Free hosting (Vercel)
- Free backend (Firebase)
- Turkish language only
- Focus on functionality first, polish later

## Key Design Philosophy
- Modern, clean interface inspired by Reddit
- Dark mode support as default
- Mobile-first responsive design
- Fast loading and interaction
- Intuitive navigation for local community
